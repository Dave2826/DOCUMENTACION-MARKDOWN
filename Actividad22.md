Conclusión
Durante este proyecto se aplicaron conceptos de álgebra lineal para manipular imágenes mediante transformaciones matriciales. Las operaciones permitieron comprender cómo la información visual puede representarse numéricamente y cómo modificaciones matemáticas producen nuevos resultados gráficos. Este proyecto fortaleció habilidades de razonamiento lógico, programación y documentación técnica.
